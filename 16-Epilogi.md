# 16. Vähemmän johtamista. Enemmän vaikutusta.

## Purpose
- Tiivistää kirjan ydinlauseeseen.
- Siirtää fokuksen uskomuksista toimintaan.
- Sulkee mallin operatiiviseen käyttöön.

## Core Claims
- Vähemmän johtamista tarkoittaa vähemmän päätösviivettä, ei vähemmän vastuuta.
- Enemmän vaikutusta syntyy, kun rakenne ohjaa päätökset vipupisteisiin.
- Palautesilmukka on tärkeämpi kuin suunnitelma.
- Resonance Leadership on jatkuva säätö, ei kertaprojekti.

## Definitions (Operational)
- **Vähemmän johtamista:** Päätösvallan hajauttamista rakenteen kautta.
- **Enemmän vaikutusta:** Mitattava muutos MaxROI-ydinmittareissa.
- **Jatkuva säätö:** Palautteen perusteella tehtävä rakenteen iterointi.

## Model / Mechanism
Kun johtaminen siirtyy rakenteisiin, päätökset syntyvät lähellä vaikutusta → palaute lyhenee → vaikutus kasvaa. Jos johtaminen keskittyy, päätös viivästyy → palautesilmukka pitenee → vaikutus heikkenee. Jatkuva säätö ylläpitää resonanssia muuttuvassa ympäristössä.

## Failure Modes
- Epilogi tulkitaan lupauksena eikä toimintakäskynä.
- Muutos nähdään projektina, ei rakenteellisena säätönä.
- Palautesilmukka jätetään huomiotta, koska ”suunta on oikea”.
- Osittainen soveltaminen: päätökset hajautetaan ilman mittareita.

## Implementation Notes
- Start here: valitse yksi päätöstyppi ja lyhennä sen palautesilmukkaa.
- Määritä, mitä ”vaikutus” tarkoittaa MaxROI-ydinmittareissa.
- Pidä päätösviive näkyvillä johtoryhmässä.
- Aseta kuukausittainen rakenteen säätösykli.
- Julkaise selkeä stop-lista päätöksille, jotka eivät tuota vaikutusta.
- Avoid this: lisätä visionääristä puhetta ilman rakenteen muutosta.
